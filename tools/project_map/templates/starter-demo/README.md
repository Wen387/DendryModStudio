# Dendry Mod Studio Starter Demo

This bundled template is copied into the user's app data before it is opened.
The copy is safe to edit, dry-run, and experiment with.

The project is intentionally tiny, but it includes the basic DendryNexus
surfaces authors expect from an SDAAH-style mod:

- a start menu,
- a sidebar/status dashboard,
- a hand scene,
- a deck scene,
- one action card,
- one pinned advisor-like card,
- small starter qualities and qdisplay text.

All content is whiteboard placeholder content. It is meant to be replaced by the
user's own mod, not edited down from an existing full game.
