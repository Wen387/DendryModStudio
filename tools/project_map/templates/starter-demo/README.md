# Dendry Mod Studio Starter Demo

This bundled template is copied into the user's app data before it is opened.
The copy is safe to edit, dry-run, and experiment with.

The project is intentionally tiny, but it is also playable. It uses a fictional
Civic Reform Office loop so mod authors can see how each Studio surface maps to
something the player can do:

- a start menu,
- a sidebar/status dashboard,
- a hand scene,
- a deck scene,
- a reusable overview card that replaces the old long start page,
- several monthly action cards that can route through the month-advance tick,
- three pinned advisor-like cards that prepare support, media pressure, budget
  capacity, and compromise routes,
- a four-event complex chain where preparation unlocks choices and changes the
  resolution,
- a monthly docket that starts in September 2032 and surfaces conditional event
  popups,
- SDAAH-like civic wire headlines in the sidebar/status dashboard,
- simple placeholder event illustrations that Studio can replace through the
  asset/image editor,
- small starter qualities and qdisplay text.

The prose intentionally includes short teaching notes. They explain what root,
hand, card, advisor, status, conditions, effects, and routes mean without
requiring the user to read Dendry source first.
