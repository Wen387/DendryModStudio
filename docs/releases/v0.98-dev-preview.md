# v0.98 Dev Preview Release Prep

Date: 2026-05-25
Status: local release-candidate preparation for a GitHub pre-release, pending final publication.

## Current Verification

- The root and desktop package metadata versions have been advanced to
  semver-compliant `0.98.0`; the user-facing release label is `v0.98.0`.
- The release workflow builds Windows `.exe`, Linux AppImage, and Linux Deb
  assets through the `Desktop Release` workflow.
- Release builds fetch the bundled Python runtime before packaging desktop
  artifacts.
- The bundled update manifest reports `latestVersion: 0.98.0`.
- v0.98.0 carries the Content Storyboard spatial canvas LOD zoom system,
  drag-to-snap card stacking with group drag, music asset editing support,
  and all v0.97.8 features.

## Release Page and Link Status

Before publishing the preview, run `Desktop Release` manually on the current
`main` with `publish_release` disabled and confirm the uploaded artifacts
include:

- `DendryModStudio-win-x64.exe`
- `DendryModStudio-linux-x64.AppImage`
- `DendryModStudio-linux-x64.deb`

After the artifact-only run passes on the current `main`, rerun the workflow
with `publish_release` enabled and `release_tag` set to `v0.98-preview.1`,
then verify the generated release page includes all three asset download links
and shows the release notes.
