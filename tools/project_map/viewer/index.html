Quick Start

Make your first safe mod proposal

Studio helps you find player-facing content, draft changes, save them as My Changes, and review what can be applied safely.

1

Open or load a project

Desktop users open a Dendry project folder. Browser users load a ProjectIndex JSON.

2

Find what to change

Use Design for player flow, or Explore for searchable lists and source evidence.

3

Create a proposal

Seed an existing Event, Card, News, or Text proposal, or start from a blank Create template.

4

Save to My Changes

Saved proposals are drafts. They do not edit source until reviewed and applied.

5

Review & Apply

Browser mode reviews plans. Desktop mode can dry-run and apply safe or guarded operations.

The demo opens as a writable copy in your app data, so you can experiment without touching a real project.

You can reopen Quick Start from More.
