# v0.98.5 Dev Preview Release Prep

Date: 2026-06-13
Status: DRAFT — version-number scaffold. Structural prep only; feature copy and
the final publish decision are pending the author's pass.

## Current Verification

- The root and desktop package metadata versions have been advanced to
  semver-compliant `0.98.5`; the user-facing release label is `v0.98.5`.
- The release workflow builds Windows `.exe`, Linux AppImage, and Linux Deb
  assets through the `Desktop Release` workflow.
- Release builds fetch the bundled Python runtime before packaging desktop
  artifacts.
- The bundled update manifest reports `latestVersion: 0.98.5`.
- v0.98.5 closes out the large-scale engineering of the v0.98 preview line.
  On the surface, it gathers the welcome experience into a full-screen Home
  Hub and introduces an illustrated What's New panel with a once-per-version
  visual introduction. Underneath, it raises the visual editor to parity with
  hand-editing mod source and lays a steadier UI/UX baseline — dark mode,
  faster large-event loads, and consistent empty / loading / error handling
  across the tool. It remains an unsigned developer preview intended for
  hands-on evaluation.
- The author hand-tested this preview end to end on the desktop build. The
  following areas were exercised in the running Electron app: Home Hub
  sections (status, announcements, templates, in-place publish with
  sync-status snapshot), the What's New panel, and the accessibility pass.

`main` with `publish_release` disabled and confirm the uploaded artifacts
include:

- `DendryModStudio-win-x64.exe`
- `DendryModStudio-linux-x64.AppImage`
- `DendryModStudio-linux-x64.deb`

After the artifact-only run passes on the current `main`, rerun the workflow
with `publish_release` enabled and `release_tag` set to `v0.98.5-preview.1`,
